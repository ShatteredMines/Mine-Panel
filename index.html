What is Mine-Panel?

It's simple. It's free. It's open-source. It's quality.

Mine-Panel is a Minecraft hosting panel for both enterprise, and home start ups! The panel is very flexible and can be easily modded to fit your needs. With new updates being released by the day - your panel will almost always be up to date! Mine-Panel is created and ran by a server owner (Craytor) which is the developer of Mine-Panel! There are multiple things that will eventually be added to the panel, but I can't do it all at a split second. That's where you can come in. Mine-Panel is still in BETA, and if you find bugs while using Mine-Panel, press "Support" in the top bar of this site. Then please briefly describe what is going on. If you want to help out, simply make a pull request.

Why should I use it?

It's actually quite simple. It's simple. It's free. It's open-source. It's quality. It's open source so any donations go right back to me to start with. Also, this panel is very light to run. It uses PHP, MySQL, Screens, and Java 7. With developments that will be released weekly you will always be up to date.
